#### Задание 2
Структура документа коллекции **books**:

```javascript
{
  title: "string",
  description: "string",
  authors: "string"
}
```

### Запросы MongoDB

Вставка данных (минимум две книги в коллекцию books)

```javascript
db.books.insertMany([
  {
    title: "Война и мир",
    description: "Роман-эпопея Льва Толстого",
    authors: "Лев Толстой"
  },
  {
    title: "Мастер и Маргарита",
    description: "Роман Михаила Булгакова",
    authors: "Михаил Булгаков"
  }
]);
```

Поиск документов по полю title

```javascript
db.books.find({ title: "Война и мир" });
```

Редактирование полей description и authors по _id

```javascript
db.books.updateOne(
  { _id: _id },
  { $set: { description: "Новое описание книги", authors: "Новый автор" } }
);
```
